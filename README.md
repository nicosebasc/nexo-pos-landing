# Nexo POS — Sitio Público

Landing page del sistema POS para comercios chilenos.

## Stack
- HTML / CSS estático
- Bootstrap Icons (CDN)
- Google Fonts — Poppins
- Design tokens sincronizados con `front-pos-comercio`

## Estructura
```
index.html         # Página principal
design-system.css  # Tokens de diseño (colores, tipografía, espaciado)
assets/            # SVGs e imágenes
```

## Deploy
GitHub Pages: rama `main`, directorio raíz.
